# Domashno-C-

#  Библиотечна Система (C++ ООП Демонстрация)

Този проект демонстрира основните принципи на **Обектно-ориентираното програмиране (ООП)** в C++, включително **капсулация**, **`const`-коректност**, **валидация на данни** и **Правилото на 5 (Rule of 5)**.

---

##  Класове и ООП Елементи

| Клас | Отговорност | Ключови ООП Елементи |
| :--- | :--- | :--- |
| `Author` | Име и година на раждане | Валидация в Setter, `const` методи. |
| `Book` | Данни за книга (заглавие, автор, цена) | **Rule of 5** (демонстрация с `= default`), **Статичен брояч** (`totalBooks`). |
| `Member` | Данни за читател | Валидация за непразен ID, конструктори. |
| `Loan` | Заемане на книга с дати и статус | Валидация на дати, метод `isOverdue`, `markReturned`. |
| `Library` | Агрегация и управление на всички обекти | Бизнес логика (`loanBook`, `returnBook`), `const` методи за търсене, **инварианти** (напр. наличност на книга). |

##  Компилиране и Изпълнение

Проектът може да бъде компилиран с g++ (GNU C++ Compiler). За да избегнете проблеми с кирилицата в конзолата, препоръчително е да използвате модерна среда или терминал, поддържащ UTF-8.

### Команда за компилиране (В случай, че всички класове са в един `main.cpp`):

```bash
g++ -std=gnu++17 -Wall -Wextra -O2 -o library main.cpp
